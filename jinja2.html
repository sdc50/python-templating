Jinja2

Jinja2 is a templating language with syntax similar to Django.

Code Sample

{% extends "layout.html" %}
  {% block body %}
  <ul>
  {% for user in users %}
    <li><a href="{{ user.url }}">{{ user.username }}</a></li>
  {% endfor %}
  </ul>
{% endblock %}
