Python Templating Systems

Jinja2

Jinja2 is a templating language with syntax similar to Django.

Code Sample

            
<!DOCTYPE html>
<html lang="en">
<head>
  <title>My Webpage</title>
</head>
<body>
  <ul id="navigation">
  {% for item in navigation %}
      <li><a href="{{ item.href }}">{{ item.caption }}</a></li>
  {% endfor %}
  </ul>
  <h1>My Webpage</h1>
  {{ a_variable }}
</body>
</html>
