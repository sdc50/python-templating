Python Templating Systems

A comparision of several templating systems for Python web development.

Overview Django Jinja2 Chameleon Mako

Mako

Mako is a non-XML syntax templating language that uses native Python. It is the default language in Pyramid and Pylons.

Code Sample

<%inherit file="base.html"/>
<%
    rows = [[v for v in range(0,10)] for row in range(0,10)]
%>
<table>
    % for row in rows:
        ${makerow(row)}
    % endfor
</table>

<%def name="makerow(row)">
    <tr>
    % for name in row:
        <td>${name}</td>\
    % endfor
    </tr>
</%def>
